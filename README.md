# Make Me Admin
Modified by Charley DAlba
Make Me Admin is a simple application for Windows that allows standard user accounts to be elevated to administrator-level, on a temporary basis.

You can find documentation in the [wiki](https://github.com/pseymour/MakeMeAdmin/wiki).

Modified source app to include Windows Hello authentication as well as autofill with ad3\userID@
Documentation for the Windows Hello reauthentication extension is in
[WINDOWS-HELLO-AUTHENTICATION.md](WINDOWS-HELLO-AUTHENTICATION.md).

Organization-specific Intune and HTTPS deployment examples are under
[Deployment](Deployment).
